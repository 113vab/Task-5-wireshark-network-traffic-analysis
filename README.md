# Task 5: Capture and Analyze Network Traffic Using Wireshark

## Overview

This task focuses on capturing and analyzing live network traffic using Wireshark, one of the most widely used network protocol analyzers. The objective is to understand how devices communicate over a network and identify different protocols involved in everyday internet activity.

---

## Objective

* Install and configure Wireshark
* Capture live network packets
* Analyze network traffic
* Identify common network protocols
* Understand packet structures and communication flow
* Gain practical experience in network monitoring

---

## Tools Used

| Tool            | Purpose                     |

| Windows 11      | Operating System            |

## Network Protocols Analyzed

### DNS (Domain Name System)

Used for translating domain names into IP addresses.

### TCP (Transmission Control Protocol)

Provides reliable communication between devices.

### TLS (Transport Layer Security)

Secures network communications through encryption.

### QUIC

Modern transport protocol used by applications and browsers for faster communication.

---

## Steps Performed

1. Installed Wireshark.
2. Selected the active Wi-Fi network adapter.
3. Started packet capture.
4. Generated network traffic by browsing websites.
5. Stopped packet capture.
6. Applied protocol filters:

   * DNS
   * TCP
   * TLS
7. Analyzed packet details.
8. Saved screenshots and findings.
